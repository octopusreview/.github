## What is Octopus?

Octopus is an AI-powered code review platform that deeply understands your codebase. It indexes your repositories, builds semantic knowledge, and delivers intelligent PR reviews with actionable findings — directly as PR comments on GitHub and Bitbucket.

## ✨ Key Features

- **🔍 AI Code Reviews** — Automated PR analysis with severity levels (🔴 Critical → 💡 Suggestion), posted as inline comments
- **💬 Chat with Your Code** — Ask questions about your codebase and get answers grounded in your actual code
- **📚 Knowledge Base** — Build organizational knowledge from docs, diagrams, and past reviews
- **📊 Team Analytics** — Track review insights, AI usage, and code quality trends across your org
- **⌨️ CLI** — Run reviews, chat with your code, and manage repos from the terminal

## How It Works

```
1. Connect  →  Link your GitHub or Bitbucket repositories
2. Index    →  Octopus embeds and indexes your entire codebase
3. Review   →  Every PR gets an AI-powered review with contextual findings
```
